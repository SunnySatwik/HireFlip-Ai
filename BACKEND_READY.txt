🎓 FAIRNESS METRICS EXPLAINED
────────────────────────────────────────────────────────────────────────────────

Demographic Parity (0-1)
  Measures if all demographic groups have equal selection rates
  Formula: min(group_rates) / max(group_rates)
  Target: Close to 1.0 (perfect fairness)

Equalized Odds (0-1)
  Measures if demographic groups have equal true positive rates
  Formula: min(group_tpr) / max(group_tpr)
  Target: Close to 1.0 (perfect fairness)

Fairness Score (0-100)
  Overall fairness rating combining multiple metrics
  Weights: Demographic Parity (40%) + Equalized Odds (40%) +
           Representation (20%)
  Target: >= 75 (good fairness)

Bias Risk Level
  Low Risk:    Score >= 75 and metrics >= 0.8
  Medium Risk: Score >= 60 and metrics >= 0.6
  High Risk:   Score < 60 or metrics < 0.6

📖 TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────────

Port 8000 already in use?
  $ uvicorn main:app --reload --port 8001

Import errors?
  $ pip install -r requirements.txt --force-reinstall

CORS issues from frontend?
  1. Verify backend is running: http://localhost:8000/health
  2. Verify frontend is on port 3000
  3. Check config.py CORS settings

CSV upload fails?
  • File must be .csv format
  • Must have required columns: name, experience, qualification, gender
  • File size must be < 10 MB
